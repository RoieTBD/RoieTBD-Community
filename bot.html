Bot

Welcome to the bot page!

The bot Greetings was made to help Discord owners to create custom "welcome" and "goodbye" messages that looks nice, instead of the boring default Discord ones.
Greetings is offering to make new default messages for that use, and the owners can decide to customize these automatically generated messages with the help of some variables.





The default "welcome" message is:

Hello {{member}}!,
welcome to {{server.name}}!


And the default "goodbye" message is:

Goodbye, {{member.name}}, we hope to see you again...

Commands & Variavles List

Commands

  • g>setWelcomeChannel [#general | general | ID]
    • Sets the welcome channel to the channel that was given with mention (#)/name/ID.
  • g>setFarewellsChannel [#general | general | ID]
    • Literally like the first command but with the channel that notifies on members who left the server.
  • g>setWelcomeMessage [text]
    • Updates the message for the members who join the server, which is sent in the welcome channel.
  • g>setFarewellsMessage [text]
    • Updates the notification message of the members who left the server, which is sent in the goodbye channel.

Variables

  • {{member}}
    • This variable expresses the mention of the member who joined the server.
  • {{member.name}}
    • The name that is displayed in the server of the member.
  • {{member.username}}
    • The username of the memeber.
  • {{member.tag}}
    • The four numbers after the hashtag in the discriminator of the member.
  • {{memeber.id}}
    • The ID of the member.
  • {{member.discriminator}}
    • The username of the member with a hashtag and then the tag of the member.
  • {{member.joinedAt}}
    • The UTC date the member joined the server.
  • {{member.createdAt}}
    • The UTC date the member created his account.
  • {{member.avatarURL}}
    • The URL of the member's avatar image, sending this will result in an image popping up.
  • {{server.name}}
    • The name of the server.
  • {{server.id}}
    • The ID of the server.
  • {{server.owner}}
    • This variable expresses the mention of the server owner.
  • {{server.owner.name}}
    • The name that is displayed in the server of the owner.
  • {{server.owner.username}}
    • The username of the owner.
  • {{server.ownerID}}
    • The ID of the owner.
  • {{server.afkChannel}}
    • This variable expresses the mention of the server's A.F.K channel.
  • {{server.afkChannelID}}
    • The ID of the A.F.K channel.
  • {{server.afkTimeout}}
    • This variable expresses the in-server setting: A.F.K Timeout.
  • {{server.createdAt}}
    • The UTC date the server has been created.
  • {{server.iconURL}}
    • The URL of the server's icon image, sending this will result in an image popping up.
  • {{server.large}}
    • According to Discord rules, this property will be false or true depending on if the server has large or small amount of members.
  • {{server.userCount}}
    • The total number of users in the server, this includes bots!
  • {{server.botCount}}
    • The number of bot users in the server.
  • {{server.memberCount}}
    • The number of members in the server.
  • {{server.nameAcronym}}
    • Do you remember when you created the server and gave it a name, and the server image was a short version of the name? Well that text is saved in here.
  • {{server.region}}
    • The server region.
  • {{server.roles}}
    • A list of the server roles. This does not include the @everyone role. Example: The roles of our server are: Community, Mod, Admin
    Notice that there is no dot at the end.
  • {{server.splashURL}}
    • The URL of the server's splash image, must be verified at Discord!
  • {{server.systemChannel}}
    • This variable expresses the mention of the default welcome & goodbye channel, make sure it is set to none so you won't have the notifications of Discord, rather ours.
  • {{server.systemChannelID}}
    • The ID of the default welcome & goodbye channel.
  • {{server.systemChannel.name}}
    • The name of the default welcome & goodbye channel.
  • {{server.verificationLevel}}
    • A number in the range 0-4 (including), this number expresses the verification level in the in-server setting: Moderation.
  • {{server.verified}}
    • True or false, depends on if the server is verified by Discord or not.
